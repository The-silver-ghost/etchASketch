# Etch A Sketch
## subtitle
This is a browser version of something between a sketchpad and a etch-a-sketch. 
This project is assigned by [The Odin Project](https://www.theodinproject.com) to test DOM manipulation skills, as well as other web development skills learned in their [Foundation](https://www.theodinproject.com/paths/foundations/courses/foundations) course. This is not a final product.
